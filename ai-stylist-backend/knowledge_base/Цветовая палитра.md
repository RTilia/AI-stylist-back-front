

# Подбор цветовой палитры

**ТАБЛИЦА  — КОНТРАСТНОСТЬ (КЛЮЧЕВАЯ ЛОГИКА)**

| Контрастность | Основание | Допустимые цвета | Запрещённые цвета | Тип образа |
| :---- | :---- | :---- | :---- | :---- |
| Низкая | кожа ≈ волосы ≈ глаза | пастель, молочный, светло-серый | чёрный, неон | мягкий, спокойный |
| Средняя | умеренная разница | средняя насыщенность | экстремальный контраст | универсальный |
| Высокая | сильная разница | чёрный, белый, насыщенные | пыльные | графичный, эффектный |

**Почему:** теория контраста (Иттен) \+ практика image consulting.

# КОНТРАСТНОСТЬ

Алгоритм работает через **шкалу светлоты от 1 до 5**  
(1 — очень светлое, 5 — очень тёмное)

## Шкала для волос

| Цвет волос | Value |
| ----- | ----- |
| Блонд | 1 |
| Пепельные / седые | 1.5 |
| Русые | 2 |
| Шатенка (коричневые) | 3 |
| Тёмно-коричневые | 4 |
| Чёрные | 5 |
|  Искусственные | не учитывает |

## Шкала для кожи

| Кожа | Value |
| ----- | ----- |
| Очень светлая | 1 |
| Светлая | 2 |
| Средняя | 3 |
| Тёмная | 4 |
| Очень тёмная | 5 |

## Шкала для глаз

| Глаза | Value |
| ----- | ----- |
| Голубые | 1 |
| Серые | 2 |
| Зелёные | 3 |
| Карие | 4 |

## РАСЧЁТ КОНТРАСТНОСТИ

Справка для определения контрастности: *глаза **всегда имеют меньший вес**, чем волосы и кожа \- это точка акцента, а не масса.*

Основной принцип

Контрастность \= **максимальная разница светлоты** между:

* волосами и кожей  
* кожей и глазами

## Интерпретация результата

| contrast score | Контрастность |
| ----- | ----- |
| 0 – 1 | Низкая |
| 2 | Средняя |
| ≥ 3 | Высокая |

Например, черные волосы (5), светлая кожа (2), голубые глаза (1)

1) 5-2 \= 3  
2) 2-1 \= 1 

3 \- высокая контрастность

* **К** — контрастность  
* **Палитра** — базовые цвета  
* **Акцент** — усиление глаз  
* **Запрет** — конфликт
